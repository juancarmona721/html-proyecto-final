# Nexcent

Este proyecto es una página web hecha con html y css  
Es un proyecto de práctica para mejorar la estructura de una página y el uso de estilos

el proyecto es solo de html y css


---

## Descripción del proyecto

se simula el sitio web de una empresa llamada Nexcent.  
Tiene varias secciones como un menú de navegación, una vista principal, información de clientes, estadísticas y un pie de página.

el objetivo es preacticar para seguir desarrollando paginas web de este estilo

---

# Archivos del proyecto

 index.html - archivo principal donde está toda la estructura de la página  
 style.css - archivo con los estilos generales  
 grid.css - archivo para organizar el contenido por columnas  
 carpeta -img/  contiene todas las imágenes usadas en la página
 pagina2.html-pagina adicional con informacion aleatoria de la pagina principal usando coneccion de links
 pagina3.html-pagina adicional con informacion aleatoria de la pagina principal usando coneccion de links



# Estructura de la página

La página está dividida en varias partes:

header / navbar  
  contiene el logo y enlaces a diferentes partes de la página.

sección principal  
  tiene un texto grande, un párrafo un botón y una imagen.

 clientes  
  muestra logos de empresas con las que se trabajo.

secciones de informacion  
  varias partes con imágenes y texto para explicar información.

Blog
  imagenes organizadas en columnas simulando publicaciones.

Footer  
  información de la empresa links cmo a redes socuales y un campo para correo.



# Estilos

 se usan dos archivos CSS para separar mejor los estilos.
 \e manejan columnas con clases como col-12 col-md-3 etc.
 se aplican colores, botones y alineación de imágenes.
 slgunos botones tienen efectos al pasar el mouse.



# Objetivo

Este proyecto fue creado para:

 practicar html y css
 aprender a organizar una página grande
 usar divs y secciones correctamente
 simular una página web real



# Estado del proyecto

el proyecto sigue en desarrollo.  
mas adelante se pueden mejorar los estilos, hacer la página responsive y agregar más funcionalidades.
